Mobile First and Responsive Site

This site is a test site I did just for fun and to experiment with mobile first and responsiveness. In the "Desktop mode", where the width of the window is >= 600px, the main navigation appears at the top right. The "mobile mode", where the width of the window is < 600px, the navigation menu appears like a tab bar at the bottom.
